# generador_arte_ia
# Generador de Arte con IA 🎨  
**Tech Stack**: Python, Stable Diffusion XL, Gradio, Hugging Face API.  
## Cómo Usarlo  
1. Escribe un prompt descriptivo (ej: "un dragón en la luna, estilo acuarela").  
2. ¡Genera arte en segundos!
